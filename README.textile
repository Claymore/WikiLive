h2. About

*WikiLive* is a special IRC client that aims to be a real time Wikipedia watch list. It connects to a Wikipedia channel on irc.wikimedia.org and presents recent changes in a nice form.

h2. Dependencies

You will need "Microsoft .NET Framework 3.5":http://www.microsoft.com/downloads/details.aspx?FamilyId=333325FD-AE52-4E35-B531-508D977D32A6&displaylang=ento to compile and use the program on Windows. You will also need "SmartIrc4net":http://smartirc4net.meebey.net/jaws/ and "SQLite":http://www.sqlite.org/index.html libraries.
